# Implementation-of-Data-Structures-In-C
Exercises Implemented

1. Expression Validator using StackImplements a function that uses a character stack to verify if an expression contains correctly nested parentheses, brackets, and braces.  The function returns 1 if the expression is correct, and 0 otherwise.
The stack is tested using either a list or an array implementation.

2. Binary Search Tree OperationsFeatures search, insertion, and deletion operations for an unbalanced binary search tree storing integers.  Includes a function named binaryTreeSearch that searches for a key and returns a pointer to the node if found, or a NULL pointer otherwise.
Involves testing both leaf-oriented and node-oriented tree implementations, along with providing a theoretical explanation of their differences.

3. Binary Tree TraversalsImplements a non-recursive level-order traversal using a queue data structure to process and print nodes from top to bottom, left to right.
The queue is implemented using either a list or an array.
Additional traversals implemented include preorder and postorder methods.

4. Tree Height and Node DepthContains a function that calculates and returns the overall height of the binary tree.
Contains a function that calculates and prints the depth of every individual node in the tree.

5. Tree Reconstruction from Traversals  Reconstructs the exact structure of a binary tree using a combination of its inorder and postorder traversals.
Reconstructs the tree using a combination of its inorder and preorder traversals.
Assumes the tree contains unique characters, allowing the traversals to be read and stored as strings.
Verifies the successful reconstruction by running a traversal function on the newly built tree.
